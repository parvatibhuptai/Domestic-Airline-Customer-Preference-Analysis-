# Domestic Airline Passenger Preferences (Statistical Inference)

A lightweight statistical analysis project using R to evaluate passenger perceptions of Indian domestic airlines (IndiGo, Akasa, Air India, etc.) across metrics like affordability, punctuality, and comfort.

## Team & My Contribution
This was a collaborative academic mini-project for a Distribution and Inference Theory course. 
* **My Role:** I assisted with the baseline data analysis using R and helped compile the final statistical report and insights. 
* **Team:** Asmita Baul, Dhruv Arora, Parvati Bhupta, Sujan Shrestha.

## Methodology & Tech Stack
* **Language/Libraries:** R Programming, `ggplot2`, `dplyr`.
* **Statistical Methods:** Extracted frequency counts for categorical survey responses to calculate variance and standard deviation.
* **Hypothesis Testing:** Conducted a One-Sample Right-Tailed Z-Test for Proportions to verify if IndiGo's market preference was statistically significant (Null probability = 0.33).

## Key Findings
* **Statistical Dominance:** The Z-test yielded a p-value of 0.0000 (p < 0.05), rejecting the null hypothesis and statistically proving IndiGo is the most preferred airline among the sample demographic.
* **Affordability:** While IndiGo dominated punctuality and service, data visualization revealed Akasa is perceived as the most affordable option.
